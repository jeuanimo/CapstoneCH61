RESPONSIVE DESIGN
================================================================================

Three main breakpoints:

Desktop (Default):    1200px and above
Tablet:              768px and below
Mobile:              480px and below

All pages include responsive design that:
- Adjusts typography sizes
- Stacks layouts vertically on mobile
- Reduces padding/margins on smaller screens
- Optimizes button and link sizes for touch

================================================================================
DARK MODE SUPPORT
================================================================================

Dark mode is automatically applied when body has class="dark-mode"

Features:
- All colors are inverted appropriately
- Form inputs use dark backgrounds
- Cards and containers adapt to dark theme
- Text contrast is maintained (WCAG AA compliant)
- No additional CSS needed - handled by existing rules

Example usage in template:
  <body class="dark-mode">
      <!-- Content automatically uses dark theme -->
  </body>

HOW TO USE IN TEMPLATES
================================================================================

BASIC TEMPLATE STRUCTURE:
==========================

{% load static %}
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
    
    <!-- Load CSS in this order -->
    <link rel="stylesheet" href="{% static 'css/base.css' %}">
    <link rel="stylesheet" href="{% static 'css/navbar.css' %}">
    <link rel="stylesheet" href="{% static 'css/pages.css' %}">
    <!-- Add page-specific CSS as needed -->
</head>
<body>  <!-- Add class="dark-mode" to enable dark theme -->
    <!-- Navigation -->
    <nav class="navbar"><!-- Navigation content --></nav>
    
    <!-- Main content -->
    <main class="main-content">
        <div class="container">
            <!-- Page content here -->
        </div>
    </main>
    
    <!-- Footer -->
    <footer><!-- Footer content --></footer>
</body>
</html>

KEY CSS CLASSES
================================================================================

BUTTONS:
  .btn            - Base button
  .btn-primary    - Blue button (default action)

  .btn-danger     - Red button (delete/danger action)

  .btn-block      - Full-width button

  .card           - Styled container with border and shadow
  .section        - Section with bottom margin

  .form-container - Form wrapper
  .form-group     - Input group with label
  .form-input     - Styled input field
  .form-errors    - Error message container
  .field-error    - Single field error message

  .alert          - Base alert
  .alert-error    - Error message (red)
