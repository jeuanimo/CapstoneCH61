How It Works
  1. Single Member: Select a specific member and create a bill for them
  2. All Members: Check "Send to All Active Members" to create the same bill for everyone in one action
  3. Bill Created: Members will see this bill in their dues page
  4. Payment Options: Members can pay online via Stripe or you can mark as paid manually
  5. Track Status: View all bills and payment status in the Dues Management page
Who Can Create Bills?
  • Admin/Superuser: Can create and manage all bills
  • Treasurer: Can create and manage chapter bills
  • Financial Secretary: Can create and track payments
  • Chapter Officers: Can create and edit bills
