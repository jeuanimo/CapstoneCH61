üìã CSV Format Guide
‚¨áÔ∏è Download Sample CSV Template

Download a ready-to-use template with example products

Required Column:
  • name - Product name (e.g., "Phi Beta Sigma T-Shirt")
Recommended Columns:
  • category - One of: apparel, accessories, drinkware, other (defaults to "other")
  • price - Price in dollars (e.g., 19.99) (defaults to 0)
Optional Columns:
  • description - Product description
  • inventory - Number of units in stock (default: 100)
  • sizes - Comma-separated sizes (e.g., "XS, S, M, L, XL")
  • colors - Comma-separated colors (e.g., "Black, White, Red")
  • image_url - Direct URL to product image (downloaded automatically)
  • image_path - Path inside ZIP file (e.g., "images/tshirt.jpg")
üì∏ Adding Images:

Option 1: Image URLs - Add an image_url column with direct image links. Images are downloaded automatically.

Option 2: ZIP File - Create a ZIP file with your images, add an image_path column with file paths (e.g., "images/product.jpg"), and upload both files together.

üí° Tips
  • Use Excel or Google Sheets to create your CSV file
  • Only the name column is required - other fields have defaults
  • Products with the same name won't be duplicated
  • Rows with errors are skipped (import continues)
  • Column names are case-insensitive ("Name" and "name" both work)
  • Maximum file size is 5MB
