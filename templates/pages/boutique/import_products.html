ðŸ“‹ CSV Format Guide
Required Columns:
  • name - Product name (e.g., "Phi Beta Sigma T-Shirt")
  • category - Must be one of: apparel, accessories, drinkware, other
  • price - Price in dollars (e.g., 19.99)
Optional Columns:
  • description - Product description
  • inventory - Number of units in stock (default: 100)
  • sizes - Comma-separated sizes (e.g., "XS, S, M, L, XL")
  • colors - Comma-separated colors (e.g., "Black, White, Red")
  • image_url - Direct URL to product image (downloaded automatically)
  • image_path - Path inside ZIP file (e.g., "images/tshirt.jpg")
ðŸ“¸ Adding Images:

Option 1: Image URLs - Add an image_url column with direct image links. Images are downloaded automatically.

Option 2: ZIP File - Create a ZIP file with your images, add an image_path column with file paths (e.g., "images/product.jpg"), and upload both files together.

Example CSV with Images:
name,category,price,description,inventory,sizes,colors,image_url

"Tumbler",drinkware,14.99,"Insulated travel mug",75,"16oz, 20oz","Black, Burgundy",https://example.com/tumbler.jpg
